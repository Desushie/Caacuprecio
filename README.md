# Caacuprecio

Proyecto web para comparar precios de productos entre distintas tiendas en Paraguay.

La idea es simple: buscar un producto y ver en qué tienda está más barato, con historial de precios, favoritos y algunas herramientas útiles para decidir mejor la compra.

---

## 🚀 Qué hace

- 🔎 Búsqueda de productos por nombre, tienda, marca o categoría  
- 🏪 Agrupación de productos similares entre distintas tiendas  
- 💲 Comparación de precios en un solo lugar  
- 📊 Historial de precios (por mes)  
- ⭐ Sistema de favoritos con cálculo de cantidades  
- 🧾 Exportar favoritos a PDF / imprimir  
- 📈 Productos más vistos / tendencias  
- 🛠 Panel admin (en desarrollo/mejoras constantes)  
- 🚩 Reporte de productos (precio incorrecto, sin stock, etc.)  

---

## Stack

- PHP (sin framework)
- MySQL / MariaDB
- Bootstrap 5
- JavaScript vanilla
- Scraping con Python (Scrapy)
